# sao-sample

> my ace SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao sample my-project
```

### From git

```bash
sao stefan de figueiredo/sao-sample my-project
```
